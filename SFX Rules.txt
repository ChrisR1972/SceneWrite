MoviePrompterAI uses lower-case parentheses ( ) exclusively for SFX.

---

1. SFX FORMAT RULE (MANDATORY)

Any text inside parentheses ( ) MUST:
- Be lower-case only
- Use underscore-separated tokens
- Match the approved SFX Whitelist
- Represent an audible sound

Parentheses must NEVER contain:
- Character names
- Locations
- Dialogue
- Narrative phrases
- Emotions or mood

---

2. CASE DISTINCTION RULE

- Characters: Capitalized Proper Names (HENRY KANE)
- Locations: Underlined (_Metro Hall_)
- Objects: Bracketed [console]
- Actions: Asterisked *grab*
- SFX: Lower-case parentheses (metal_clang)

Case is semantically meaningful and must be preserved.

---

3. NO IDENTITY CREATION RULE

SFX:
- Do NOT create characters
- Do NOT create objects
- Do NOT create vehicles
- Do NOT create environments

SFX only annotate sound.

---

4. VALIDATION PASS (REQUIRED)

After scene generation:
- Scan all parentheses
- Reject any entry that is:
  - Capitalized
  - Multi-word prose
  - Not in the SFX Whitelist
- Replace invalid entries or remove them

---

EXPECTED RESULT:

- Zero SFX/character collisions
- Clean parsing
- Deterministic audio timelines
- Screenplay-style scene output
